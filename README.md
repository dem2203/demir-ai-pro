## ğŸš€ Quick Start

### Prerequisites

- Python 3.11+
- PostgreSQL 15+
- Binance API keys
- Telegram bot (for alerts)

### Local Setup

```bash
# Clone repository
git clone https://github.com/dem2203/demir-ai-pro.git
cd demir-ai-pro

# Create virtual environment
python -m venv venv
source venv/bin/activate # Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Configure environment
cp .env.example .env
# Edit .env with your credentials
```

### Configuration (.env)

```bash
# Binance
BINANCE_API_KEY=your_api_key
BINANCE_SECRET_KEY=your_secret_key

# Telegram
TELEGRAM_BOT_TOKEN=your_bot_token # From @BotFather
TELEGRAM_CHAT_ID=your_chat_id # From @userinfobot

# Database
DATABASE_URL=postgresql://user:pass@localhost:5432/demir_ai

# Trading
TRADING_MODE=PAPER # PAPER or LIVE
DEFAULT_SYMBOL=BTCUSDT
ACCOUNT_BALANCE=10000
```

### Run Trading System

```bash
# Main app (includes dashboard + AI engine)
python main.py

# Access dashboard
open http://localhost:8000/trading-terminal
```

### Dashboard Features

- Real-time P&L tracking
- Win rate & profit factor

- WebSocket auto-updates
